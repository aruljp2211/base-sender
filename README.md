# Base Wallet Sender 🔵

Simple Python tool to send ETH on Base network.

## 🚀 Features
- Send ETH to another wallet
- Lightweight CLI
- Fast & low gas fee

## 🧠 Network
- Base Mainnet
- Chain ID: 8453

## ⚙️ Usage
1. Install web3
   pip install web3

2. Run script
   python bws.py

## ⚠️ Note
Use at your own risk. Never share your private key.
